# eLisaBet — Техническое задание

Шуточная внутренняя платформа для ставок на ЧМ 2026 между командами (отделами) компании.

## Стек

- **Frontend/Backend:** Next.js (App Router), TypeScript, Tailwind CSS
- **База данных:** Supabase (PostgreSQL)
- **Деплой:** Vercel + Supabase
- **Данные матчей:** [football-data.org](https://www.football-data.org/) (бесплатный tier)
- **Цвета:** тёмно-синий `#0f2744` + оранжевый `#f97316`

## Участники и авторизация

- До ~10 команд-участников
- Вход по **уникальному коду команды** (без email/пароля)
- При первом входе команда **самостоятельно задаёт название**
- Сессия хранится в защищённой cookie
- Один суперадмин (отдельный секретный ключ в env)

## Типы ставок

| Тип | Описание | Очки | Дедлайн |
|---|---|---|---|
| Исход матча | Победа хозяев / победа гостей (ничьи нет) | +1 | До kick-off |
| Точный счёт | Финальный счёт на табло (включая пенальти) | +2 (+1 за исход = **+3** суммарно) | До kick-off |
| Победитель ЧМ | Одна ставка на страну-победителя | +5 | До старта 1/16 |
| 3-е место | Одна ставка на страну, займёт 3-е место | +3 | До старта 1/16 |

### Ограничения

- На каждый матч: **одна ставка на исход** + **одна на точный счёт**
- На победителя и 3-е место: **по одной ставке** на команду, можно менять до дедлайна
- Ставки принимаются только на матчи **плей-офф с 1/16** (Round of 32) и далее, включая матч за 3-е место и финал
- Вариант «Ничья» в форме **не показывается**

## Приватность

- Ставки других команд **не видны**
- Команда видит только **свои** ставки и общий лидерборд

## Лидерборд

- Сортировка по сумме очков (убывание)
- При равенстве очков — больше угаданных **точных счетов** выше

## Парсинг матчей

- Фоновая синхронизация с football-data.org (cron на Vercel)
- Сохранение: расписание, статус, счёт, этап турнира
- После финала матча — автоматический пересчёт очков
- Fallback: админ может вручную править счёт и статус матча

## Экраны

1. **Вход** — ввод кода команды
2. **Первый вход** — выбор названия команды
3. **Матчи** — список матчей плей-офф, формы ставок
4. **Спецставки** — победитель ЧМ и 3-е место
5. **Мои ставки** — история и текущие ставки команды
6. **Лидерборд** — рейтинг всех команд
7. **Админка** — создание кодов, ручная правка матчей, пересчёт очков
